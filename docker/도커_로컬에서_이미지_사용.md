# 로컬에서 도커 이미지를 사용하기

<br />
<br />

* 폐쇄망에서는 도커 이미지를 어떻게 옮길까?

---

```
도커 허브가 없는 환경에서는
tar파일로 파일 자체를 옮겨야 한다.

로컬에서 사용하는 예시를 알아보자.
```

<br />
<br />
<br />
<br /> 

1. 도커 이미지 만들기

```zsh
docker build -t 이미지이름:태그 .
# docker build -t next-app:latest .
```  

<br />
<br />
<br />

2. 이미지 -> tar으로 빼기

```zsh
docker save -o 파일.tar 이미지이름:태그
# docker save -o next-app.tar next-app:latest
```

<br />
<br />
<br />

3. tar 불러오기 — Docker 있을 때

```zsh
sudo k3s ctr images import 파일.tar
# sudo k3s ctr images import next-app.tar
```

<br />
<br />
<br />

4. tar 불러오기 — Docker 없을 때 (k3s)

```zsh
sudo k3s ctr images import 파일.tar
# sudo k3s ctr images import next-app.tar
```
